Documentation

1)Installation
- Copier le répertoire "ns_monmodule" dans le répertoire "modules" de Prestashop
- Activer le module "Module trouve ta zone" depuis l'administration de Prestashop

2)Utilisation
- Créer des catégories qui correspondent aux activitées des commerçants
- Créer des produits et les lier avec les différentes catégories
- Dans Prestashop vous devez renseigner :
     - le nom du produit
     - donner une reference pour indiquer la zone ex: 75020
     - dans les metadonnées mettre juste le lien d'une page ex: https://monmarche.drindrin.fr/collections/les-pains-dazor
